Constant current buck-boost converter based on the TPS63070 and INA180. The TPS63070 is a constant voltage buck-boost converter, so I'm using an INA180 (similar to this app note: https://www.ti.com/lit/an/slva419d/slva419d.pdf?ts=1689743420901) to convert it to a constant current converter.
